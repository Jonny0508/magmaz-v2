# Magmaz v2

Magmaz v2 es una versión mejorada de mi proyecto original, creado con Next.js y Tailwind CSS. Esta versión implementa una estructura de carpetas profesional y está lista para escalar.

## 🚀 Tecnologías utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Git](https://git-scm.com/)
- [GitHub](https://github.com/)
